# Todo-Application
A simple To-Do Application built using HTML, CSS and JavaScript.  Users can add tasks, delete tasks and mark tasks as completed.

## Features
- Add new tasks
- Delete tasks
- Mark tasks as completed
- Responsive design

## Technologies Used
- HTML5
- CSS3
- JavaScript

## How to Use
1. Type a task in the input box
2. Click "Add New Task" button
3. Click on task text to mark as completed
4. Click to delete a task
